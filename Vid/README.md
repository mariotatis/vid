# Vid - iOS Video Player

A simple iOS video player built with SwiftUI.

## Features
- **Library**: Lists videos from a selected directory.
- **Playlists**: Create playlists and add videos.
- **Player**: Full screen playback with Loop and Shuffle support.

## Usage
1. Go to **Library** tab.
2. Tap **Add** to import videos from files (mp4, mov, m4v).
3. Videos are imported to the library.
